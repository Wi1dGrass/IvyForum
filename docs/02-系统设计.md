## 4. 统一约定

### 响应体
```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

### 分页响应
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "records": [],
    "total": 100,
    "page": 1,
    "size": 20
  }
}
```

### 错误码段
- 1xxx：认证 / 授权错误
- 2xxx：参数校验错误
- 3xxx：业务错误
- 5xxx：服务端错误

## 5. 前端路由结构

```
/                          首页（最新帖子列表）
/channel/:id                频道页
/tag/:id                    标签页
/search?keyword=            搜索结果
/hot                        热门榜
/post/:id                   帖子详情
/post/new                   发帖
/post/edit/:id              编辑帖子
/login                      登录
/register                   注册
/user/:id                   个人主页
/profile                    编辑个人资料
/notifications              通知中心
/admin                      管理后台
├ /admin/users             用户管理
├ /admin/channels          频道管理
├ /admin/tags              标签管理
├ /admin/reports           举报管理
└ /admin/posts             帖子管理
```

路由守卫：
- meta.requiresAuth：未登录跳 /login
- meta.roles：角色不符跳 403

## 6. 构建与部署
- 后端：`mvn clean package`，`java -jar backend/target/school-forum.jar`
- 前端：`npm install && npm run build`，dist 部署到 nginx，反向代理 /api → 后端
- 开发期前端 Vite proxy /api → http://localhost:8080
