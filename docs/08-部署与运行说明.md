# 08 部署与运行说明

## 1. 环境要求

| 组件 | 版本 | 路径 / 备注 |
|------|------|------|
| JDK | 17.0.8 | `D:\javase\jdk-17.0.8` |
| Maven | 3.6+ | 系统可用 |
| MySQL | 5.7.26 | `D:\phpstudy_pro\Extensions\MySQL5.7.26`，root / 123456 |
| Node | 22.21.1 | npm 10.9.4 |
| 浏览器 | Chrome / Edge 最新版 | — |

## 2. 初始化数据库

```bash
# 使用 mysql 客户端执行
mysql -uroot -p123456 < backend/sql/init.sql
```
脚本会：
- 创建数据库 `school_forum`
- 创建 11 张表
- 插入初始数据（管理员 admin / admin123、5 频道、10 标签）

## 3. 后端启动

```bash
# 设置 JDK（如系统默认不是 17）
set JAVA_HOME=D:\javase\jdk-17.0.8
set PATH=%JAVA_HOME%\bin;%PATH%

# 编译
cd backend
mvn clean package -DskipTests

# 运行
java -jar target/school-forum-1.0.0.jar

# 或开发模式
mvn spring-boot:run
```

- 默认端口：8080
- Swagger：http://localhost:8080/api/doc.html
- 配置文件：`backend/src/main/resources/application.yml`
- 测试配置：`backend/src/main/resources/application-test.yml`

## 4. 前端启动

```bash
cd frontend
npm install
npm run dev      # http://localhost:5173，proxy /api -> http://localhost:8080
# 或构建
npm run build    # 产物 dist/
npm run preview
```

## 5. 生产部署
- 后端：`java -jar school-forum-1.0.0.jar`，配合 systemd / nohup
- 前端：将 `dist/` 放入 nginx 静态目录，配置反向代理：

```nginx
server {
  listen 80;
  server_name your.domain;
  root /var/www/school-forum/dist;
  location / { try_files $uri $uri/ /index.html; }
  location /api/ {
    proxy_pass http://127.0.0.1:8080/api/;
    proxy_set_header Host $host;
  }
}
```

## 7. 常见问题
- **MySQL 连接失败**：确认 phpstudy 启动 MySQL 且端口 3306 未占用
- **JWT 401**：本地存储清空重新登录
- **端口冲突**：修改 application.yml 的 server.port 和 vite.config.ts 的 server.port
