## 3. Axios 封装
- baseURL: `/api`
- 请求拦截：携带 `Authorization: Bearer <token>`
- 响应拦截：
  - res.code !== 0 → ElMessage.error(message)
  - 401 → 清空 token，路由跳 /login
  - 返回 res.data（剥离外层）

## 4. Pinia Store

### userStore
- state: token, userId, username, nickname, role, avatar
- actions: login, logout, fetchMe, updateProfile
- getters: isLoggedIn, isAdmin

### notificationStore
- state: unreadCount
- actions: fetchUnread, markRead

## 5. 关键页面设计

### 首页 / 频道页
- 左侧：频道树导航
- 右侧顶部：排序切换（最新 / 最热）+ 标签筛选条
- 右侧主体：PostCard 列表 + 分页
- 右上方"发帖"按钮（登录可见）

### 帖子详情
- 标题 + 作者信息 + 标签
- 正文：v-md-editor 预览模式渲染 Markdown
- 底部：点赞 / 收藏按钮
- 评论树：按 root_id 分组，回复使用楼中楼折叠

### 发帖页
- 频道下拉选择
- 标签多选（最多 5）
- v-md-editor 编辑
- 提交 → 跳详情页

### 个人主页
- 顶部头像 / 昵称 / 角色徽章 / 签名
- Tab 切换：发帖 / 收藏 / 点赞

### 通知中心
- 列表 + 一键已读
- 顶部未读红点（轮询 30s）

### 管理后台
- AdminLayout 侧栏菜单 5 项
- 各页面 Element Plus Table + 分页 + 弹窗表单

## 6. 开发命令
```bash
cd frontend
npm install
npm run dev      # 开发，端口 5173，proxy /api -> 8080
npm run build    # 构建 dist
npm run preview
```
